# RSS DOWNLOADER

A Node.js script that, given a CSV with RSS feed URLs, reads and downloads the corresponding images of the day.

## HOW TO RUN DEVELOP MODE 

``` bash
npm i
```
``` bash
npm run dev
```
## HOW TO RUN PRODUCTION
``` bash
npm install --production
```
``` bash
npm run start
```
